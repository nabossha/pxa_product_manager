Installation
==============

- Install the **Products manager** Extension in the Extension Manager
- Include the Template by selecting **Template** and the root of your site. Then click on **Edit the whole template record**
- In the Include tab select **Products manager (pxa_product_manager)** and **Products manager: Solr Configuration (pxa_product_manager)**  on the right side of the Include static (from extensions): selection box.
